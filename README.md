# ProposalAI

A standalone AI-powered proposal generator. No framework, no build step — just one HTML file.

## Setup

1. Deploy to Vercel, Netlify, or any static host
2. Open the app and enter your Anthropic API key in the generator
3. Start generating!

## Files

- `index.html` — the entire app
- `vercel.json` — routing config for Vercel
